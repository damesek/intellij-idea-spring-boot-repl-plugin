  1. Bridge + Agent JAR
      - Futtasd a saját gépeden:

        cd ~/Documents/Codes/sb-repl
        mvn -f sb-repl-bridge/pom.xml clean install -Dgpg.skip=true
        mvn -f sb-repl-agent/pom.xml clean install -Dgpg.skip=true
        Ez létrehozza a JAR-okat a sb-repl-bridge/target és sb-repl-agent/target könyvtárban.
        Ez létrehozza a JAR-okat a sb-repl-bridge/target és sb-repl-agent/target könyvtárban.
      - Másold át a két modul tartalmát (vagy akár az egész sb-repl mappát) ZIP-ben a másik Mac-re. Ott elég mvn -f sb-repl-bridge/pom.xml install -Dgpg.skip=true és mvn -f sb-repl-
  agent/pom.xml install -Dgpg.skip=true, így az új gép ~/.m2 cache-ébe is bekerülnek.
  2. Spring app
      - A pom.xml-be vedd fel a bridge-et:

        <dependency>
            <groupId>hu.baader</groupId>
            <artifactId>sb-repl-bridge</artifactId>
            <version>0.5.0-SNAPSHOT</version>
        </dependency>
        és Gradle-ben mavenLocal()-t tartsd bent a repositories között.
  3. IntelliJ plugin
      - A ./gradlew buildPlugin utáni ZIP-et (pl. build/distributions/sb-repl-1.0-SNAPSHOT.zip) vidd át a másik gépre, és lehet telepíteni (File → Settings → Plugins → Install from
  Disk). A plugin új verziója már automatikusan keresi a sb-repl-agent JAR-t a ~/.m2-ben.
